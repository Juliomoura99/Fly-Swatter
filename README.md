# 🪰 Fly Swatter Game

A **Fly Swatter Game** developed with **HTML, CSS, and JavaScript**.

The goal of the game is simple: the player must click on the flies that appear randomly on the screen before they disappear. Each missed fly costs the player one life. The game also includes difficulty level selection, a countdown timer, and a life system.

Have fun trying to swat all the flies before time runs out! 😄

---

## 📌 About the Project

This project was developed to practice fundamental **JavaScript** concepts, DOM manipulation, click events, time control, dynamic element creation, and basic game logic.

The game simulates a simple and fun experience where flies appear in random positions on the screen, and the player must click on them to remove them.

---

## 🎮 Features

This project includes the following features:

- Difficulty level selection
- Random fly generation on the screen
- Different fly sizes
- Random fly image direction
- Life system
- Countdown timer
- Victory screen
- Game over screen
- Click interaction
- Fly removal when clicked
